### Installation


# Clone the repository


```bash
$ git clone https://github.com/n1ckdevops/python-project-49
```

Use make (Makefile):


make install

```bash
	poetry install
```

make build

```bash
	poetry build
```

make publish

```bash
	poetry publish --dry-run
```

make package-install

```bash
	python3 -m pip install --user --force-reinstall  dist/*.whl
```
